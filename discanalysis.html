21 Discriminant Analysis

So far we have considered Discriminant Analysis (DA) from a largely conceptual standpoint. In this lecture, we start to formalize our notions into a mathematical framework in what we will call Probabilistic Discriminant Analysis.

21.1 Probabilistic DA

We start with the notion of the Bayes Classifier, which is explained in more detail in section 2.2.3 of ISL. Suppose we have a response with \(K\) classes \((C_1, \dots, C_K)\), and further suppose we have \(p\) predictors. The Bayes Classifier gives us the following classification rule:

\[ \text{assign } x_i \text{ to the class for which } Pr( \{ y_i = k \} \mid x_i) \text{ is the largest} \]

This rule is optimal in the sense that it minimizes the misclassification error, though it is usually computationally intensive (since it involves computing probabilities for all \(K\) classes). As such, we would like to develop a slightly more feasible classification rule.

We start by examining the other conditional probability: that is, \(Pr(x_i \mid y_i = k)\). Of course, these two conditional probabilities are related:

\[ Pr(x_i \mid \{ y_i = k \} ) = \frac{Pr(x_i \cap \{ y_i = k\} ) }{Pr( \{ y_i = k \} ) } \]

\[ Pr(\{ y_i = k \} \mid x_i ) = \frac{Pr( \{ y_i = k\} \cap x_i ) }{Pr( x_i ) } \]

Note that the numerator in the above equations are the same: hence, we obtain (notice that we omit writing in the braces in the probabilities, for convenience):

\[ Pr(y_i = k \mid x_i) Pr(x_i) = Pr(y_i = k) Pr(x_i \mid y_i = k) \]

after rearranging terms, yields

\[ \underbrace{ Pr(y_i = k \mid x_i) }_{\text{posterior}} = \frac{ \overbrace{ Pr(x_i \mid y_i = k) }^{\text{likelihood}} \overbrace{ Pr(y_i = k) }^{\text{prior}} }{Pr(x_i) } \]

This is nothing but Bayes’ Rule! Now, we’re almost done; we just need to simplify the denominator a bit. Typically, to compute the denominator, we would use partitioning:

\[ Pr(x_i) = \sum_{k=1}^{K} Pr(y_i = k) Pr(x_i \mid y_i = k) \]

and we would write:

\[ Pr(y_i = k \mid x_i) = \frac{Pr(x_i \mid y_i = k) Pr(y_i = k) }{Pr(x_i) } \]

We can re-express the above equation with more traditional notation:

\[ Pr(y_i = k \mid x_i) = \frac{\pi_k f_k(x) }{\sum_{k=1}^{K} \pi_k f_k(x) } \]

where \(\pi_k\) denotes the priors and \(f_k(x)\) denotes the class conditional (aka likelihood) probabilities.

21.2 Estimations

Let’s parse through our equation from above. Clearly, in order for this to be of any practical use, we need to estimate some of these quantities. Estimating \(\pi_k\) is relatively intuitive: \(\hat{\pi}_k = n_k / n\) where \(n_k = |C_k|\) denotes the size of class \(k\) and \(n\) denotes the total number of data points.

Now, here comes one of the key assumptions of Discriminant Analysis: we assume that our class-conditional probabilities follow a Gaussian distribution.

21.2.1 Univariate Data (\(\boldsymbol{p = 1}\))

In the case of univariate data, we have

\[ f_k(x) = \frac{1}{\sqrt{2 \pi \sigma_k^2}} \hspace{1mm} \exp\left\{ -\frac{1}{2} \left( \frac{x - \mu_k}{\sigma_k} \right)^2 \right\} \]

21.2.2 Multivariate Data (\(\boldsymbol{p > 1}\))

In the case of multivariate data, we use a multivariate normal distribution as our likelihood:

\[ f_k(\mathbf{x}) = \frac{1}{(2 \pi)^{p/2} | \mathbf{\Sigma_k} |^{1/2} } \exp\left\{ - \frac{1}{2} (\mathbf{x} - \boldsymbol{\mu_k})^{\mathsf{T}} \mathbf{\Sigma_k}^{-1} (\mathbf{x} - \boldsymbol{\mu_k}) \right\} \]

where \(\mathsf{\Sigma_k}\) denotes the variance-covariance matrix associated with \(C_k\), and \(\boldsymbol{\mu_k}\) denotes the centroid of \(C_k\). Note that the exponent of the likelihood is simple the Mahalanobis distance between \(\mathbf{x}\) and \(\boldsymbol{\mu_k}\).

21.2.3 Multivariate Estimations

For \(\hat{\mu}\), we can use the centroid of \(C_k\); i.e. the average individual of class \(k\). For \(\mathbf{\widehat{\Sigma}_k}\), we can use the within-variance matrix:

\[ \mathbf{\widehat{\Sigma}_k} = \frac{1}{n-1} \mathbf{X_k}^\mathsf{T} \mathbf{X_k} \]

(assuming mean-centered data within each class).

21.3 Discriminant Functions

Given all of our above estimations, we can now find an estimate for \(Pr(y_i = k \mid x_i)\), using equation (1) above (for the moment, let us consider the multivariate case):

\[ \widehat{ Pr(y_i = k \mid \mathbf{x} ) } = \frac{\hat{\pi}_k \hat{f}_k(\mathbf{x}) }{\sum_{k=1}^{K} \hat{\pi}_k \hat{f}_k(\mathbf{x}) } \]

Now, note that the denominator remains constant across classes. Furthermore, we have a Gaussian expression in our numerator; hence, it makes sense to take logarithms:

\[ \ln\left[ Pr(y_i = k \mid \mathbf{x} ) \right] \propto \ln( \hat{\pi}_k) + \ln\left[ \hat{f}_k(\mathbf{x}) \right] \]

We now substitute the Multivariate Normal p.d.f. into our expression above, to see:

\[\begin{align*} \ln\left[ \hat{f}_k(x) \right] & = \ln \left[ (2\pi)^{-p/2} | \mathbf{\hat{\Sigma}_k} |^{-1/2} \exp\left\{ - \frac{1}{2} (\mathbf{x} - \boldsymbol{\hat{\mu}})^{\mathsf{T}} \mathbf{\hat{\Sigma}}^{-1} (\mathbf{x} - \boldsymbol{\hat{\mu}}) \right\} \right] \\ & \to \ln\left( | \mathbf{\hat{\Sigma}_k} |^{-1/2} \right) - \frac{1}{2} (\mathbf{x} - \boldsymbol{\hat{\mu}})^{\mathsf{T}} \mathbf{\hat{\Sigma}}^{-1} (\mathbf{x} - \boldsymbol{\hat{\mu}}) \\ & = -\frac{1}{2} \ln\left( | \mathbf{\hat{\Sigma}_k} | \right) - \frac{1}{2} \left[ \mathbf{x}^\mathsf{T} \mathbf{\hat{\Sigma}_k^{-1}} \mathbf{x} - 2 \mathbf{x}^\mathsf{T} \mathbf{\hat{\Sigma}_k^{-1}} \boldsymbol{\hat{\mu}} + \boldsymbol{\hat{\mu}}^{\mathsf{T}} \mathbf{\hat{\Sigma}_k^{-1}} \boldsymbol{\hat{\mu}} \right] \end{align*}\]

Substituting this back into our expression for \(\ln[Pr(y_i = k \mid \mathbf{x})]\) leads us to the so-called discriminant functions:

\[ \delta_k(\mathbf{x}) := -\frac{1}{2} \ln\left( | \mathbf{\hat{\Sigma}_k} | \right) - \frac{1}{2} \left[ \mathbf{x}^\mathsf{T} \mathbf{\hat{\Sigma}_k^{-1}} \mathbf{x}^\mathsf{T} - 2 \mathbf{x}^\mathsf{T} \mathbf{\hat{\Sigma}_k^{-1}} \boldsymbol{\hat{\mu}} + \boldsymbol{\hat{\mu}}^{\mathsf{T}} \mathbf{\hat{\Sigma}_k^{-1}} \boldsymbol{\hat{\mu}} \right] + \ln\left( \hat{\pi}_k \right) \]

In general, our classification rule is as follows:

\[ \text{assign } \mathbf{x} \text{ to the class for which } \delta_k(\mathbf{x}) \text{ is the largest} \]

21.4 Quadratic Discriminant Analysis (QDA)

Let us classify the order of each term, w.r.t. \(\mathbf{x}\):

\[ \delta_k(\mathbf{x}) := -\frac{1}{2} \underbrace{ \ln\left( | \mathbf{\hat{\Sigma}_k} | \right) }_{\text{constant}}- \frac{1}{2} \left[ \underbrace{ \mathbf{x}^\mathsf{T} \mathbf{\hat{\Sigma}_k^{-1}} \mathbf{x} }_{\text{quadratic}} - \underbrace{ 2 \mathbf{x}^\mathsf{T} \mathbf{\hat{\Sigma}_k^{-1}} \boldsymbol{\hat{\mu}} }_{\text{linear}} + \underbrace{ \boldsymbol{\hat{\mu}}^{\mathsf{T}} \mathbf{\hat{\Sigma}_k^{-1}} \boldsymbol{\hat{\mu}} }_{\text{constant}} \right] + \underbrace{ \ln\left( \hat{\pi}_k \right) }_{\text{constant}} \]

Of course, we could group terms to obtain an expression of the form \(\text{const} + \text{linear} + \text{quadratic}\). That is, we obtain a quadratic function of \(\mathbf{x}\); this leads us to Quadratic Discriminant Analysis (QDA).

21.5 Linear Discriminant Analysis

What if, in our expression for \(\delta_K(\mathbf{x})\), we have that $ = = = =: $? In this case, the discriminant function becomes

\[ \delta_k(\mathbf{x}) = \underbrace{ \ln\left( \hat{\pi}_k \right) }_{\text{constant}} - \underbrace{ \frac{1}{2} \ln\left( | \mathbf{\hat{\Sigma}} | \right) }_{\text{no k dependency}} - \frac{1}{2} \left[ \underbrace{ \mathbf{x}^\mathsf{T} \mathbf{\hat{\Sigma}^{-1}} \mathbf{x} }_{\text{no k dependency}} - \underbrace{ 2 \mathbf{x}^\mathsf{T} \mathbf{\hat{\Sigma}^{-1}} \boldsymbol{\hat{\mu}} }_{\text{linear}} + \underbrace{ \boldsymbol{\hat{\mu}}^{\mathsf{T}} \mathbf{\hat{\Sigma}^{-1}} \boldsymbol{\hat{\mu}} }_{\text{constant}} \right] \]

In other words, ignoring terms that do not depend on \(k\), we obtain (again, w.r.t. \(\mathbf{x}\))

\[ \delta_k(\mathbf{x}) = \underbrace{ \ln\left( \hat{\pi}_k \right) }_{\text{constant}} - \frac{1}{2} \left[ - \underbrace{ 2 \mathbf{x}^\mathsf{T} \mathbf{\hat{\Sigma}^{-1}} \boldsymbol{\hat{\mu}} }_{\text{linear}} + \underbrace{ \boldsymbol{\hat{\mu}}^{\mathsf{T}} \mathbf{\hat{\Sigma}^{-1}} \boldsymbol{\hat{\mu}} }_{\text{constant}} \right] \]

That is, our discriminant function is now linear w.r.t. \(\mathbf{x}\); hence, we end up with Linear Discriminant Analysis (LDA) in its most general form.

21.6 Canonical Discriminant Analysis

Let us continue our idealistic assumptions. We again assume that $ = = = =: $ as well as \(\hat{\pi}_1 = \dots = \hat{\pi}_k =: \hat{\pi}\). The

discriminant function then becomes

\[ \delta_k(\mathbf{x}) = \underbrace{ \ln\left( \hat{\pi} \right) }_{\text{no k dependency}} + \underbrace{ \mathbf{x}^\mathsf{T} \mathbf{\hat{\Sigma}^{-1}} \boldsymbol{\hat{\mu}} }_{\text{linear}} - \frac{1}{2} \underbrace{ \boldsymbol{\hat{\mu}}^{\mathsf{T}} \mathbf{\hat{\Sigma}^{-1}} \boldsymbol{\hat{\mu}} }_{\text{constant}} \]

which means, after ignoring \(k\)-independent terms, we have

\[ \delta_k(\mathbf{x}) = \underbrace{ \mathbf{x}^\mathsf{T} \mathbf{\hat{\Sigma}^{-1}} \boldsymbol{\hat{\mu}} }_{\text{linear}} - \frac{1}{2} \underbrace{ \boldsymbol{\hat{\mu}}^{\mathsf{T}} \mathbf{\hat{\Sigma}^{-1}} \boldsymbol{\hat{\mu}} }_{\text{constant}} \]

Now, what is going in with this first term? It is simply the distance between \(\mathbf{x}\) and the centroid of \(C_k\), using the within-class variance matrix as a metric matrix. In other words, the above expression is our good friend CDA!

21.7 Naive Bayes

Again assume that \(\mathbf{\hat{\Sigma}_1} = \mathbf{\hat{\Sigma}_2} = \dots = \mathbf{\hat{\Sigma}_K} =: \mathbf{\hat{\Sigma}}\) as well as \(\hat{\pi}_1 = \dots = \hat{\pi}_k =: \hat{\pi}\). Now, we also assume that \(\mathbf{\hat{\Sigma}}\) is diagonal; i.e. that the predictors are uncorrelated:

\[ \mathbf{\hat{\Sigma}} = \begin{pmatrix} Var(X_1) & 0 & \dots & 0 \\ 0 & Var(X_2) & \dots & 0 \\ \vdots & \vdots & \ddots & \vdots \\ 0 & 0 & \dots & Var(X_p) \\ \end{pmatrix} \]

This leads to what is known as Naive Bayes.

21.8 Fifth Case

Again assume that \(\mathbf{\hat{\Sigma}_1} = \mathbf{\hat{\Sigma}_2} = \dots = \mathbf{\hat{\Sigma}_K} =: \mathbf{\hat{\Sigma}}\) as well as \(\hat{\pi}_1 = \dots = \hat{\pi}_k =: \hat{\pi}\). Also assume that \(\mathbf{\hat{\Sigma}} = \mathbf{I_p}\), the \(p \times p\) identity matrix. In other words, our Mahalanobis distance collapses to the Euclidean distance.
